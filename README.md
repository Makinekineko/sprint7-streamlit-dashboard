# Sprint 7 — Streamlit Dashboard

Este proyecto consiste en la creación y despliegue de una aplicación web interactiva usando **Streamlit** para explorar un dataset en formato CSV mediante visualizaciones.

##  Funcionalidad
- Visualización de datos con **histogramas**.
- Gráficos de **dispersión** interactivos.
- Selección dinámica de columnas numéricas.
- Aplicación desplegada en la nube con **Render**.

##  Tecnologías utilizadas
- Python
- pandas
- Streamlit
- Plotly
- Render
- Git & GitHub
